# Task Management Desktop App

A personal productivity desktop application built with Java and JavaFX for task organization and daily workflow management.

---

## 🧠 Architecture

The application follows a layered architecture:

- **app** → Application bootstrap and context
- **controller** → UI logic & user interaction handling
- **service** → Business logic layer
- **repository** → Data access abstraction
- **database** → SQLite database management
- **model** → Domain entities
- **view** → FXML views & CSS styling

This separation ensures scalability, maintainability, and clean code structure.

---

## 🚀 Features

- Task creation & management
- Study tracking
- Workout tracking
- Dashboard overview
- SQLite local database persistence
- Modular MVC-inspired architecture
